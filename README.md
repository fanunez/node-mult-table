# Opciones

Primer programa en Node :)

```
Options:
      --help     Show help                                      [boolean]
      --version  Show version number                            [boolean]
  -b, --base     Base for multiplication                        [number]  [required]
  -l, --list     Show the result on screen                      [boolean] [default: false]
  -u, --until    Multiplicate the base until reach this limit   [number]  [required]
```
